
# Music Salon Management System

## Description
Информационная система «Музыкальный салон» предназначена для автоматизации учета дисков, исполнителей и продаж в музыкальном магазине. Приложение позволяет управлять каталогом продукции, хранить изображения обложек в базе данных и формировать аналитическую отчетность по выручке и складским остаткам. Система поддерживает многопользовательский доступ с предварительной аутентификацией администратора.

## Installation
Для локальной установки приложения выполните следующие шаги:
1. Убедитесь, что у вас установлен компилятор `gcc` и библиотека `sqlite3`.
2. Склонируйте репозиторий с исходным кодом:
   `git clone https://github.com/CheVitel/TPMP_Lab4_Cherepkovskiystelchenko.git`
3. Перейдите в папку проекта: `cd TPMP_Lab4_Cherepkovskiystelchenko`
4. Соберите проект с помощью Makefile:
   `make`
5. База данных `salon.db` будет создана автоматически при первом запуске.

## Usage
1. Запустите скомпилированный файл: `./bin/music_salon`
2. Введите учетные данные (по умолчанию: login `admin`, password `admin123`).
3. Используйте цифровое меню для навигации:
   - `1` — Добавление нового диска.
   - `2` — Просмотр всего инвентаря.
   - `3` — Вывод отчета по остаткам.
   - `4` — Анализ выручки по авторам.
